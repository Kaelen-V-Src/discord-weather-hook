# Discord Weather Hook

A lightweight Discord bot that fetches real-time weather updates and posts them via webhooks.

## Features
- Real-time weather data from OpenWeatherMap API
- Discord webhook integration for automatic updates
- Configurable location and update frequency
- Lightweight and easy to deploy

## Installation
1. Clone the repository
2. Install dependencies: npm install
3. Configure config.json with your Discord webhook URL and OpenWeatherMap API key
4. Run: node index.js

## Configuration
Edit config.json with your settings:
- discordBotToken: Your Discord bot token
- weatherAPIKey: Your OpenWeatherMap API key
- location: Target location for weather updates
